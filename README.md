# 🌾 AI-Based Crop Recommendation System

## 📌 Overview

This project is a **Machine Learning-based Crop Recommendation System** that suggests the most suitable crop to grow based on soil nutrients and environmental conditions.

It uses features like **Nitrogen (N), Phosphorus (P), Potassium (K), Temperature, Humidity, pH, and Rainfall** to predict the best crop.

---

## 🎯 Objectives

* Analyze agricultural data using data science techniques
* Build a classification model for crop prediction
* Visualize dataset using graphs
* Evaluate model performance using metrics

---

## 📊 Dataset

The dataset contains:

* 2200 rows
* 8 columns

### Features:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

### Target:

* Crop label

---

## 🧠 Machine Learning Model

* Algorithm: **Random Forest Classifier**
* Type: Classification
